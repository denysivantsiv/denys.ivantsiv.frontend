position: absolute

Елемент позиціонується відносно найближчого позиціонованого предка. Якщо такого немає - відносно вікна браузера.

absolute

position: fixed

Елемент позиціонується відносно вікна браузера і не рухається при прокрутці.

fixed

position: relative

Елемент зміщується відносно свого звичайного положення.

relative

position: static

Звичайне положення елемента у потоці (значення за замовчуванням).

static
